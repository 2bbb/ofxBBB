# ofxBBB

## How to use

### clone

```bash
git clone https://github.com/2bbb/ofxBBB.git /path/to/oF/addons/
git submodule update --init --recursive
```

### load addon

with ProjectGenerator

### use

```cpp
#include "ofxBBB.h"
```

or

```cpp
#include <bbb/bit_by_bit.hpp>
```

### notice

this addon only set the include path to `bit_by_bit`
